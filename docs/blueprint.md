# **App Name**: Mandate Navigator

## Core Features:

- Mandate List View: Display a sortable and filterable list of UN mandate documents, each linked to their corresponding UN entity.
- Filtering Options: Implement filters for UN entity, year of publication, and keywords within the document.
- Operative Paragraph Highlighting: Scan mandate documents and highlight operative paragraphs, for each selected mandate.

## Style Guidelines:

- Primary color: Forest green (#3B822B), symbolizing international cooperation and growth.
- Background color: Very light green (#F2F8F0). The green hue evokes internationality, and it's desaturated to sit quietly in the background.
- Accent color: Earthy orange (#C75300), drawing inspiration from humanitarian assistance.
- Headline font: 'Belleza', sans-serif. Body font: 'Alegreya', serif. This pairing brings a sense of personality, while assuring that longer texts are very readable.
- Use a clean and structured layout, prioritizing clear information hierarchy and easy navigation.
